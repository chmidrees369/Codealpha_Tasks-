# Hangman Game 🎮

Welcome to the **Hangman Game**, a simple text-based Python game where the player guesses a hidden word **one letter at a time**.  

This project is built for the **CodeAlpha Python Internship** to practice Python fundamentals like `random`, `while loop`, `if-else`, `strings`, and `lists`.

---

## 📝 Features

- Randomly selects a word from a **predefined list**.
- Player has **6 chances** to guess wrong letters.
- Displays the **current word progress** with blanks (`_`) for unguessed letters.
- Validates input to accept **only one alphabet letter** at a time.
- Console messages for **wrong guesses** and **game status**.
- Shows **win or lose message** at the end.

---

## 💻 How to Play

1. Clone or download the repository.
2. Open terminal or command prompt.
3. Run the game:

```bash
python hangman_game.py
